# react-jianshu

> 使用 React 实现简书首页

### 技术栈

**React + React-Router + Redux(MobX) + TypeScript**

### 构建

```bash
# install dependencies
yarn

# serve with hot reload at localhost:3000
yarn start

# build for production env
yarn build

# serve for production env
node prod.server.js
```
